# Reporting programming challenge

Given an org chart listing managers and their reports and a list of
pairs of employee names, compute the least common manager for each
pair.

## Sample data

 * names - list of 1000 names, one per line
 * reports - list of managers and their reports, one manager per line,
   names separated by commas, the first name on each line is the
   manager
 * pairs - list of pairs of names separated by a comma, one pair per line.
